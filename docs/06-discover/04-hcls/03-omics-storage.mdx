---


If user forgets the password to the application,
User can always reset the password using 'Forgot password' link in the
login page. Click on the 'Forgot password' link in the login page and
'Forgot Password' page will be opened.


![Forgot Password](https://media-hub.amorphicdata.io/docs/media/authentication/v2_forgot_password.png)

Enter the 'Username' or 'Email ID' associated with the user account and
click on 'Reset Password' button.

![Enter Username](https://media-hub.amorphicdata.io/docs/media/authentication/v2_add_username.png)

A verification code will be sent to
the specified email address and 'Reset Password' page will be displayed.
Enter the 'Verification Code' and the 'New Password/Confirm Password'
details and click on 'Submit' button.

![Reset Password](https://media-hub.amorphicdata.io/docs/media/authentication/v2_verification_code.png)

Password will be successfully changed and the login page will be displayed on clicking continue.

![Password Changed](https://media-hub.amorphicdata.io/docs/media/authentication/v2_reset_password.png)
